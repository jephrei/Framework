The Grid

The grid system is based on fractions. 1/2, 2/3, 5/8, etc. They can be nested, used in any combination, reversed in mobile, with or without gutters. There's no margin trickery here either. That means a column with the class .span-1-2 is exactly 50% wide.

Rows with gutters (.row.spaced) have a negative margin on each side and applies a padding to each of it's children. So pseudo classes or extra .first and .last type classes aren't necessary.

Typography

The font families chosen are serif, sans-serif, and monospace...that's it. Browsers and the operating systems they live in pick the best looking font they can if it's not specified. The font sizes are em based so they scale with the body font-size percentage which, in turn, changes with media queries.

Headers

The header sizes have been reset so that h6 is equal in size to the body font size. Dangerous? Probably. It's an experiement so i may undo it depending on how it works in practice.
